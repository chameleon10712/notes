InnoDB
========

InnoDB，是MySQL的數據庫引擎之一，為MySQL AB發行binary的標準之一。InnoDB由Innobase Oy公司所開發，2006年五月時由甲骨文公司併購。與傳統的ISAM與MyISAM相比，InnoDB的最大特色就是支援了ACID相容的事務（Transaction）功能，類似於PostgreSQL。

目前InnoDB採用雙軌制授權，一是GPL授權，另一是專有軟件授權。
