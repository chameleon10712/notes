==============================
Relational database 關聯式資料庫
==============================
- MySQL
- MariaDB
- PostgreSQL

是建立在關聯模型(Relational model)基礎上的資料庫，藉助於集合代數(Algebra of sets)等數學概念和方法來處理資料庫中的資料
