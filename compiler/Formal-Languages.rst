Formal Languages
==================

**Def.**

Let ``Σ`` be a set of characters (an alphabet).
A language over ``Σ`` is a set of strings of characters drawn from ``Σ``

--

|

Meaning function ``L`` maps syntax to semantics

.. code::

  L(e) = m


- e: regular expression
- m: set of strings
